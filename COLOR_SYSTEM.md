# ServicePro Color System

## Brand Colors

### Primary Brand - Deep Indigo
- **Color**: `#360185`
- **Usage**: Headers, Navbar, Footer, Main headings, Logo accents, Primary text
- **Components**: Navigation bar background, section headings, footer background

### Primary Action - Royal Magenta
- **Color**: `#8F0177`
- **Usage**: Primary buttons, Active states, Links, CTA actions
- **Components**: All primary buttons, link hover states, icon backgrounds

### Alert/Highlight - Crimson Rose
- **Color**: `#DE1A58`
- **Usage**: Alerts, Destructive actions, Badges, Warning indicators
- **Components**: Delete buttons, error states, alert badges

### Accent - Golden Amber
- **Color**: `#F4B342`
- **Usage**: Premium indicators, Icon accents, Highlights (very limited)
- **Components**: Logo icon background, CTA button in dark sections, accent text

## Neutral Colors

- **White**: `#FFFFFF` - Main background
- **Light Gray**: `#F9FAFB` - Card backgrounds, light backgrounds
- **Medium Gray**: `#E5E7EB` - Borders, dividers
- **Text Primary**: `#1F2937` - Body text
- **Text Secondary**: `#6B7280` - Secondary text, descriptions
- **Text Muted**: `#9CA3AF` - Helper text, captions

## Design Principles

1. **No Gradients**: All colors are solid, single colors
2. **High Contrast**: Ensure text is always visible against backgrounds
3. **Professional**: Clean, minimal aesthetic
4. **Consistent**: Same color combinations used throughout
5. **Accessibility**: WCAG AA compliant color contrast ratios

## Usage Tips

- Use `#360185` for major UI elements (header, footer, headings)
- Use `#8F0177` for all interactive elements (buttons, links, actions)
- Reserve `#DE1A58` for destructive/alert actions only
- Limit `#F4B342` to premium accents and logo elements
- Keep neutral colors at 70% of design
- Brand colors at 30% combined (20% indigo, 7% magenta, 2% rose, 1% amber)
